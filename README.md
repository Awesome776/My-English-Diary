# My-English-Diary
This is a repository for pure English diary. The purpose is to practice daily English writing skills and typing speed while recording my daily learning progress and experiences. My teacher requires us to improve our typing speed, so I was inspired to create this repository while practicing on typing software. However, due to my limited English proficiency, I will write the first few entries in Chinese first, then have AI translate them into English before copying them in. I will proceed step by step, hoping that in the future I can rely on myself to write a complete English diary. For any words I don't know at all, I will list them separately below for memorization and bold them in the diary. Additionally, I will include a famous quote at the end of each diary.
